# 🚗 AI 小车 - 树莓派 4B 全向控制系统

基于麦克纳姆轮的 AI 视觉小车主控程序。

## 📋 硬件接线指南

### L298N 电机驱动 (#1 - 前轮)

| 树莓派 GPIO | L298N #1 | 功能 |
|:-----------:|:--------:|:----:|
| BCM 17 | IN1 | 前左轮方向 A |
| BCM 18 | IN2 | 前左轮方向 B |
| BCM 12 | ENA (PWM) | 前左轮速度 |
| BCM 22 | IN3 | 前右轮方向 A |
| BCM 23 | IN4 | 前右轮方向 B |
| BCM 13 | ENB (PWM) | 前右轮速度 |

### L298N 电机驱动 (#2 - 后轮)

| 树莓派 GPIO | L298N #2 | 功能 |
|:-----------:|:--------:|:----:|
| BCM 24 | IN1 | 后左轮方向 A |
| BCM 25 | IN2 | 后左轮方向 B |
| BCM 19 | ENA (PWM) | 后左轮速度 |
| BCM 5  | IN3 | 后右轮方向 A |
| BCM 6  | IN4 | 后右轮方向 B |
| BCM 26 | ENB (PWM) | 后右轮速度 |

> ⚡ **电源**: L298N 使用**外接电源**（6-12V），不要从树莓派取电！
> 树莓派 GPIO 的 3.3V 直接接 L298N 逻辑输入即可（L298N 兼容 3.3V 逻辑电平）。

### HY-SRF05 超声波

| 树莓派 GPIO | 模块 | 说明 |
|:-----------:|:----:|:----:|
| BCM 27 | TRIG | 触发脚 (OUT) |
| BCM 16 | ECHO | 回响脚 (IN) |

> ⚠️ **必须分压**: ECHO 输出 5V，必须用电阻分压（1kΩ + 2kΩ）降到 3.3V 再接 GPIO 16！

### 云台舵机

| 树莓派 GPIO | 舵机 | 说明 |
|:-----------:|:----:|:----:|
| BCM 20 | PAN (水平) | 左右转动 |
| BCM 21 | TILT (俯仰) | 上下转动 |

> 舵机电源可以从树莓派 5V 取电（两个 SG90 没问题），也可以外接 5V 电源。

### 摄像头 & 音频

| 设备 | 接口 | 说明 |
|:----:|:----:|:----:|
| CSI 摄像头 | CSI-2 排线接口 | 主摄像头，通过云台舵机控制方向 |
| USB 摄像头 | USB 2.0/3.0 | 副摄像头，带麦克风 |
| 小喇叭 | 3.5mm 音频口 | TTS 语音输出 |

## 🚀 快速开始

### 1. 启动 pigpio 守护进程（舵机用）

```bash
sudo pigpiod
```

### 2. 一键启动小车

```bash
cd ~/ai-car
bash start.sh
```

### 3. 打开控制台

在电脑或手机的浏览器打开: **http://树莓派IP:5000**

## 🛠️ 常见问题

**Q: GPIO 权限错误**
```
A: 确保以非 root 用户运行，RPi.GPIO 需要属于 gpio 组
   sudo usermod -a -G gpio $USER
```

**Q: pigpio 连接失败**
```
A: 先启动守护进程: sudo pigpiod
```

**Q: 摄像头画面不显示**
```
A: 检查:
   - CSI 摄像头排线是否插紧
   - 启用摄像头: sudo raspi-config → Interface Options → Camera
   - USB 摄像头设备: ls /dev/video*
```

**Q: 舵机抖动的厉害**
```
A: 检查供电是否充足，不要从 GPIO 3.3V 取电给舵机。
   改用外接 5V 电源给舵机供电，共地即可。
```
